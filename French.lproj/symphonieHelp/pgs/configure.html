Configuration de symPhonie

symPhonieest simple à configurer.

Configuration du compte SIP
Dans la version actuelle, la configuration symPhonie est disponible pour les services SIP du fournisseur de services Free (Freephonie).
D'autres fournisseurs de service SIP, et également une configuration générique, seront ajoutés dans les prochaines versions.

Configuration de symPhonie pour Freephonie
Réglages son dans Préférences Système
Verifiez que dans les Preferences Système, la fonction "réduction du bruit de fond" est activée sur le microphone.
Détection du réseau "home"
"Home detection" est une fonctionalité future, qui n'est pas encore disponible. symPhonie pourra détecter si vous êtes chez vous, et changer automatiquement la configuration en conséquence.
Par exemple, dans une configuration standard, vous pourrez recevoir les appels entrant sur la freebox lorsque vous êtes chez vous, mais sur votre Mac lorsque vous êtes en déplacement.
Dans cette version, seul le réglage "Recevoir les appels sur le Mac" est disponible. C'est la valeur par défaut, au démarage, de la destination des appels entrants, qui peut être changée dans la fenêtre principale.
Configuration Audio
symPhonie peut modifier les niveaux d'entrée et de sortie audio lorsqu'on reçoit ou lance un appel téléphonique. En général, on souhaite un volume plus élevé lors d'un appel téléphonique que, par exemple, lorsqu'on écoute de la musique. Cochez "réglages de volume spécifique pour symPhonie" pour activer cette fonction.
Le réglage "amplification supplémentaire sur le micro" permet d'augmenter le niveau d'entrée du micro au dela du maximum. Cela peut être nécessaire avec le microphone intégré.
Les réglages "Entrée" et "Sortie" permettent de choisir les périphériques audio utilisés (par exemple un casque)
Configuration sonnerie
La configuration de la sonnerie permet de choisir plusieurs modes de notification quand un appel est reçu  symPhonie peut afficher une fenêtre "popup" transparante, jouer une sonnerie audio, etc...
De plus, symPhonie peut mettre en pause les applications multimédia telles que Lecteur DVD, iTunes, VLC, etc.. lorsqu'un appel est reçu ou lancé.
note: dans la version actuelle, il n'est pas possible de changer de sonnerie.
Configuration divers
Le réglage "Carnet d'adresse visible au démarage" permet de choisir si, par défaut, la fenêtre Carnet d'adresse est visible lorsque l'on est dans l'état "prêt"
En cochant "double click sur nom ou numéro", on peut appeler directement, sans confirmation, en double cliquant sur le nom ou le numéro dans le panneau carnet d'adresse.
